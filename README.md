# Express-pat
Express-pat, my personal Express JS scaffolding (Mongodb integration using mongoose + Cloudrun ready with Dockerfile)

```
express-pat
 ┣ controllers
 ┃ ┗ UserController.js
 ┣ middlewares
 ┃ ┗ auth.js
 ┣ models
 ┃ ┗ User.js
 ┣ routes
 ┃ ┣ api.js
 ┃ ┗ index.js
 ┣ .env-template
 ┣ .gitignore
 ┣ Dockerfile
 ┣ README.md
 ┣ app.js
 ┣ package-lock.json
 ┗ package.json
 ```

## Tutorial
Rename .env-template to .env

Run

```
npm install
```

```
node app.js
```
